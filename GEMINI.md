# Project Rules

This is a React + Vite + Tailwind + Express + Prisma ecommerce admin dashboard.
Tech Stack:
- React
- Vite
- Tailwind
- Express
- Prisma
- PostgreSQL

Rules:
- Preserve DashboardLayout architecture
- Never duplicate Navbar or Sidebar
- Preserve dark SaaS design
- Use responsive Tailwind
- Avoid hook-order bugs
- Avoid overflow issues
- Do not rewrite unrelated files
- Prefer modular reusable components
- Preserve existing APIs/routes
- Use defensive rendering
- Keep UI premium and production-grade
